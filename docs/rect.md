Rect Object Type
================

`Rect` accepts numeric pixel values to describe how far to extend a rectangular area. These values are added to the original area's size to expand it.

Example[​](#example "Direct link to Example")
---------------------------------------------

js

```
{  
    bottom: 20,  
    left: null,  
    right: undefined,  
    top: 50  
}  

```
